# FridgeChef AI App

**FridgeChef AI** is an Android app designed to help users discover recipes based on the ingredients available in their fridge and their nutritional goals. With AI-powered recommendations, voice-guided cooking steps, live auto-complete suggestions, and seamless cloud syncing ensures data consistency across multiple devices, the app offers a personalized cooking experience tailored to users' health and dietary preferences. üçΩÔ∏èü§ñ

### Setup and Installation üñ•Ô∏è

1. Install [Android Studio](https://developer.android.com/studio).
2. Clone this repository.
3. Open the project in Android Studio.
4. Add API keys for Spoonacular, Gemini, and Google Custom Search to `local.properties`:
   ```
   API_KEY_spoonacular=<Your_Spoonacular_API_Key>
   API_KEY_gemini=<Your_Gemini_API_Key>
   API_KEY_custom_search=<Your_Google_Custom_Search_API_Key>
   ```
5. Sync Gradle files.
6. Run the application on an emulator or physical device.
